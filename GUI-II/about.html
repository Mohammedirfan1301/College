Ski Maps is a semester long group project for the GUI II class at UMass Lowell. The project aims to make it easier for skiers and snow boarders to find out what the current conditions are like at a given mountain. The project sources data for six different ski resorts, and plots the current trail conditions on a trail map. Sourcing this data is accomplished via a Python script that scapes each ski resorts snow report page and saves the data in JSON format.
